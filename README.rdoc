= BSUIR Schedule Complete

The most relevant deployed version: http://bsuir-schedule-complete.herokuapp.com

== DESCRIPTION:

A simple BSUIR schedule parser that provides a transparent and eye-pleasing display.

== FEATURES:

Generates group schedule for specified group and subgroup using BSUIR Schedule API (http://www.bsuir.by/schedule/API.xhtml). Provides PDF file for download.

== CREDITS:

* Using Nogokiri for XML parsing: https://github.com/sparklemotion/nokogiri
* Using WickedPDF for PDF generation: https://github.com/mileszs/wicked_pdf
* Using WickedPDF binary for Heroku: https://github.com/rposborne/wkhtmltopdf-heroku
